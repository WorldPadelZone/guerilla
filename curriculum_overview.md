# Guerilla AI Training — 12-Month Curriculum
**January 2026 Edition**

## The Structure

**Month 1** = Introduction to the "Art of Possible" (Image + Video Generation overviews)  
**Months 2-12** = Deep dives into specific tools + prompt guides

## Monthly Deliverables

Each month includes:
1. **Module HTML** — Content document (like `foundations.html`)
2. **Prompt Guide** — 10-20 battle-tested prompts for the month's tools
3. **Homework Assignment** — Practical task using real client work
